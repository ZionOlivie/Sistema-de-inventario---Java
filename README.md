Sistema Básico de Inventário em Java (Feature 1 - TP1)

Este projeto é uma aplicação de console em Java desenvolvida como parte da Feature 1 (TP1). O objetivo é estabelecer e aplicar conceitos iniciais da linguagem aprendidas em aula e na trilha do Instituto Infnet.
O sistema simula um cadastro e consulta de produtos simples em um inventário.


Como Executar o Projeto
Para rodar esta aplicação em sua máquina:

1.  Requisitos: Você deve ter o JDK 21 instalado.
2.  Copie o Repositório:
3.  Abra o IntelliJ IDEA -> Importe o projeto no IntelliJ
4.  Execute a classe principal: InventarioApp.
5.  O programa iniciará no console com o menu de opções.
